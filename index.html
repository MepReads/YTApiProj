Mep Presents: The Mep Reads API

Have you ever rendered a video for multiple hours? If so, you might have done that over night or while being at work. The problem is, you also have to upload it to YouTube afterwards, which takes a lot of time again. To immediately start the upload process after completing rendering, I created the Mep Reads Api. This gives you more Time to put into Video Creation and Quality.

Please note: The Mep Reads Api isn't publically available. It's use is only allowed for the creator (me) and friends.
